# Seção 05 — Microsoft Azure (Azure)

Este documento resume a aula **“Conhecendo a Microsoft Azure”** e organiza os pontos principais para revisão rápida e estudo prático.

## Objetivos de aprendizagem

- Entender o que é a Microsoft Azure e seu papel no mercado de cloud.
- Conhecer o portal da Azure e como iniciar o acesso à plataforma.
- Identificar serviços essenciais de computação, armazenamento, rede e identidade.
- Registrar os conceitos iniciais para exploração prática da plataforma.

## Arquivos desta seção

A pasta contém o seguinte material:

- `0_Conhecendo_A_Microsoft_Azure.pdf` — aula introdutória sobre a Azure, seus conceitos básicos e primeiras observações de uso.

## Resumo da aula

A Microsoft Azure é a plataforma de computação em nuvem da Microsoft, com serviços para infraestrutura, plataforma e soluções gerenciadas.

O material destaca o uso da nuvem para provisionar recursos sob demanda e a importância de conhecer o portal da Azure para navegação e configuração inicial.

### Pontos principais

- A Azure é a plataforma de cloud da Microsoft.
- O acesso é feito pelo **Azure Portal**.
- A plataforma oferece serviços de computação, armazenamento, rede, banco de dados e identidade.
- A integração com o ecossistema Microsoft é um diferencial importante para ambientes corporativos.

## Visão rápida

A Azure oferece recursos para hospedar máquinas virtuais, aplicações web, bancos de dados, redes virtuais e serviços de segurança. Nesta aula introdutória, o foco é apresentar a plataforma e seus primeiros passos.

## Principais serviços da Azure

- Computação:
  - **Virtual Machines** — máquinas virtuais na nuvem.
  - **Azure App Service** — hospedagem gerenciada para aplicações web.
  - **Azure Functions** — execução serverless baseada em eventos.
  - **Azure Kubernetes Service (AKS)** — orquestração gerenciada de containers.

- Armazenamento e bancos de dados:
  - **Azure Blob Storage** — armazenamento de objetos.
  - **Azure Files** — compartilhamento de arquivos gerenciado.
  - **Azure SQL Database** — banco relacional gerenciado.
  - **Cosmos DB** — banco NoSQL globalmente distribuído.

- Rede e entrega:
  - **Virtual Network (VNet)** — redes privadas isoladas.
  - **Load Balancer** — distribuição de tráfego.
  - **Azure DNS** — gerenciamento de DNS.

- Segurança e identidade:
  - **Microsoft Entra ID** — identidade e controle de acesso.
  - **Azure Key Vault** — gerenciamento de segredos e chaves.
  - **Microsoft Defender for Cloud** — postura de segurança e recomendações.

## Sugestões de estudo e prática

1. Acesse o portal oficial da Azure e navegue pela página inicial.
2. Explore os serviços em destaque no catálogo da plataforma.
3. Compare a proposta da Azure com AWS e GCP em termos de ecossistema e integração.
4. Anote quais serviços seriam úteis para aplicações corporativas no seu cenário.

## Referência

- Microsoft Azure: https://azure.microsoft.com

# Section 05 — Microsoft Azure (Azure)

This document summarizes the lesson **“Getting to Know Microsoft Azure”** and organizes the main points for quick review and practical study.

## Learning objectives

- Understand what Microsoft Azure is and its role in the cloud market.
- Learn how to access the Azure Portal and get started with the platform.
- Identify essential compute, storage, networking, and identity services.
- Capture the initial concepts needed for hands-on exploration of the platform.

## Files in this section

This folder contains the following material:

- `0_Conhecendo_A_Microsoft_Azure.pdf` — introductory lesson about Azure, its basic concepts, and first usage notes.

## Lesson summary

Microsoft Azure is Microsoft’s cloud computing platform, with services for infrastructure, platform, and managed solutions.

The material highlights using the cloud to provision resources on demand and the importance of knowing the Azure Portal for navigation and initial setup.

### Key points

- Azure is Microsoft’s cloud platform.
- Access is provided through the **Azure Portal**.
- The platform offers compute, storage, networking, database, and identity services.
- Integration with the Microsoft ecosystem is an important advantage for enterprise environments.

## Quick overview

Azure provides resources to host virtual machines, web applications, databases, virtual networks, and security services. In this introductory lesson, the focus is on presenting the platform and its first steps.

## Main Azure services

- Compute:
  - **Virtual Machines** — cloud virtual machines.
  - **Azure App Service** — managed hosting for web applications.
  - **Azure Functions** — event-driven serverless execution.
  - **Azure Kubernetes Service (AKS)** — managed container orchestration.

- Storage and databases:
  - **Azure Blob Storage** — object storage.
  - **Azure Files** — managed file sharing.
  - **Azure SQL Database** — managed relational database.
  - **Cosmos DB** — globally distributed NoSQL database.

- Networking and delivery:
  - **Virtual Network (VNet)** — isolated private networks.
  - **Load Balancer** — traffic distribution.
  - **Azure DNS** — DNS management.

- Security and identity:
  - **Microsoft Entra ID** — identity and access control.
  - **Azure Key Vault** — secrets and key management.
  - **Microsoft Defender for Cloud** — security posture and recommendations.

## Study and practice suggestions

1. Open the official Azure portal and browse the home page.
2. Explore the highlighted services in the platform catalog.
3. Compare Azure with AWS and GCP in terms of ecosystem and integration.
4. Note which services would be useful for enterprise applications in your scenario.
